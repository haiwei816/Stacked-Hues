# 游戏核心场景与 2.5D 架构

## 1. 游戏视角与环境
- **相机设置**：使用 2.5D 等距视角 (Isometric View)，相机必须设置为正交投影 (Orthographic)，角度通常为俯视 45 度、侧转 45 度（类似于经典的城市建造类游戏视角），玩家可以通过键盘鼠标的方式改变视角。
- **环境底座**：一个现代微缩街区底座 (Diorama block)。底座包含柏油马路、人行道和微缩的行道树。

## 2. 核心大楼结构 (U型管的具象化)
- **双子楼设计**：在街区上矗立着两栋现代摩天大楼（左塔楼和右塔楼）。大楼的正面必须是透明的玻璃幕墙，以便玩家能清晰看到内部，两栋楼使用L型的拐角连接两栋楼的一楼，这个拐角的横截面也是透明可见的，从而与两栋大楼共同构成一个完整的 **“U型物理容器”**。

## 3. 基础游戏循环 (Game Loop)
- **输入检测**：玩家点击 [左大楼顶部天台] 或 [右大楼顶部天台]。
- **生成事件**：点击后，在对应的天台正上方生成一个巨大的、具有高弹物理特性的 **彩色果冻球 (Jelly Ball)**，并在重力作用下砸进大楼内部。
